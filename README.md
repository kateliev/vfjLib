# vfjLib

Low-level reader and writer for FontLab JSON (VFJ) source font files

### Installation

```
pip install --user git+https://github.com/kateliev/vfjLib
```

### Usage

```python
import vfjLib
jfont = vfjLib.vfjFont(path_to_font)
```
